**1. Create `ui/column_selector.py`:**

Function `render_column_selector(sheets: dict[str, pd.DataFrame], detected: dict[str, list[str]]) -> None`:

- Takes sheets and detected sensitive columns from detector
- Creates tabs per sheet via `st.tabs(list(sheets.keys()))`
- For each sheet tab:
  - "Выбрать все" and "Снять все" buttons in a `st.columns([1, 1, 4])` row:
    - "Выбрать все" sets `st.session_state[f"cb_{sheet}_{col}"] = True` for all columns, then `st.rerun()`
    - "Снять все" sets `st.session_state[f"cb_{sheet}_{col}"] = False` for all columns, then `st.rerun()`
    - Use unique button keys: `key=f"sel_all_{sheet}"`, `key=f"desel_all_{sheet}"`
  - For each column in df.columns:
    - Use `st.columns([0.6, 0.2, 0.2])` for layout: [checkbox | badge | type_toggle]
    - Checkbox: `st.checkbox(col, value=default_val, key=f"cb_{sheet}_{col}")` where default_val is `st.session_state.get(f"cb_{sheet}_{col}", col in detected.get(sheet, []))`
    - Badge: call `classify_column_type(col, df[col])` to get type. Render `st.markdown(f"<span style='background:#e8f4f8;border-radius:4px;padding:2px 6px;font-size:0.8em'>[текст]</span>", unsafe_allow_html=True)` for text, same with `background:#fff3e0` for numeric
    - Type toggle (ONLY for numeric-dtype columns): `st.selectbox("Тип", ["коэффициент", "идентификатор"], key=f"type_{sheet}_{col}", index=0 if classify returns "numeric" else 1, label_visibility="collapsed")`. This lets user switch numeric columns between coefficient and identifier masking. Text columns do NOT get a toggle.

def render() -> None:
    st.header("Маскирование данных")

    if stage is None:

    elif stage == STAGE_UPLOADED:

Update `_render_step_preview()`:
- Change `disabled=True` to `disabled=False` on "Далее" button
- On click: set `st.session_state[STAGE] = STAGE_COLUMNS`, call `st.rerun()`

Add `_render_step_columns()`:
- `render_steps(current=2)`
- `sheets = st.session_state[SHEETS]`
- `detected = detect_sensitive_columns(sheets)` (recompute — fast and stateless)
- Call `render_column_selector(sheets, detected)`
- Two-column layout for buttons:
  - "Назад" button -> set STAGE to STAGE_UPLOADED, st.rerun()
  - "Замаскировать" primary button -> on click:
    1. Read checkbox states from session_state: `selected = {}` for each sheet, for each col, check `st.session_state.get(f"cb_{sheet}_{col}", False)`
    2. Build mask_config: for each selected col, determine type: if numeric dtype and `st.session_state.get(f"type_{sheet}_{col}") == "коэффициент"` -> "numeric", else -> "text". Text-dtype columns always "text".
    3. Call `masked_sheets, mapping, stats = mask_sheets(sheets, mask_config)`
    4. Store in session state: `st.session_state[MASKED_SHEETS] = masked_sheets`, `st.session_state[MAPPING] = mapping`, `st.session_state[STATS] = stats`, `st.session_state[MASK_CONFIG] = mask_config`
    5. Set `st.session_state[STAGE] = STAGE_MASKED`, `st.rerun()`
  - If no columns selected (all unchecked), show `st.warning("Выберите хотя бы одну колонку для маскирования")` and do NOT proceed

Add `_render_step_masked()`:
- `render_steps(current=3)`
- `masked_sheets = st.session_state[MASKED_SHEETS]`
- `stats = st.session_state[STATS]`
- `file_name = st.session_state.get(FILE_NAME, "файл")`
- `st.subheader(f"Результат маскирования: {file_name}")`
- Statistics row: `col1, col2 = st.columns(2)`, `col1.metric("Замаскировано значений", stats["masked_values"])`, `col2.metric("Уникальных сущностей", stats["unique_entities"])`
- Call `render_preview(masked_sheets)` to show masked data preview (reusing existing widget)
- Disabled download stubs: `st.button("Скачать замаскированный файл", disabled=True)`, `st.button("Скачать маппинг (JSON)", disabled=True)`, `st.button("Скачать маппинг (Excel)", disabled=True)` with `st.caption("Будет доступно в следующей версии")`
- Navigation buttons in two columns:
  - "Назад к выбору колонок" -> clear MASKED_SHEETS, MAPPING, STATS from session_state (but preserve checkbox states), set STAGE to STAGE_COLUMNS, st.rerun()
  - "Сбросить" -> clear ALL session state keys (SHEETS, RAW_BYTES, STAGE, FILE_NAME, SELECTED_COLUMNS, MASK_CONFIG, MAPPING, MASKED_SHEETS, STATS, and all `cb_*` and `type_*` keys), st.rerun(). To clear dynamic keys: `keys_to_clear = [k for k in st.session_state if k.startswith(("cb_", "type_"))]`
