   - `_normalize(value: str) -> str`: NFC normalization, strip, upper, remove all quote variants (""«»'"), collapse whitespace
   - `_derive_prefix(col_name: str) -> str`: Take the last meaningful word from column name. Rules: split by spaces, skip service words ["имя", "наименование", "название", "номер"], take the remaining last word and capitalize first letter. Fallback: first word. Examples: "Имя предприятия" -> "Предприятие", "Автор изменения" -> "Автор" (skip none, take second-to-last meaningful word — actually just take last noun). Implementation: split col_name, filter out SKIP_WORDS = ["имя", "наименование", "название", "номер", "внеш.", "внешний"], return last remaining word with .capitalize(). If nothing remains, return first word of col_name with .capitalize().
   - `_index_to_label(n: int) -> str`: 1->A, 2->B, ..., 26->Z, 27->AA (Excel-column style, as in RESEARCH.md)
   - `def build_text_mapping(sheets, mask_config) -> dict[str, str]`: Single-pass: iterate all sheets, all text-type columns from mask_config, collect unique normalized values, assign "Prefix A/B/C" pseudonyms with global counters dict per prefix. Return {normalized_value: pseudonym}.
   - `def build_numeric_mapping(sheets, mask_config) -> dict[str, float]`: For each unique column name with type "numeric" across all sheets, generate one `random.uniform(0.5, 1.5)` coefficient. Return {col_name: coefficient}.
   - `def apply_text_masking(series, mapping) -> pd.Series`: Vectorized via `series.map(lambda v: mapping.get(_normalize(str(v)), v) if pd.notna(v) else v)`
   - `def apply_numeric_masking(series, multiplier) -> pd.Series`: `result = series * multiplier`. If `pd.api.types.is_integer_dtype(series)`, `result.round().astype("Int64")`, else `result.round(2)`.
   - `def mask_sheets(sheets, mask_config) -> tuple[dict[str, pd.DataFrame], dict, dict]`:
     1. Build text_mapping via build_text_mapping
     2. Build numeric_mapping via build_numeric_mapping
     3. For each sheet, deep-copy df, apply text masking to text columns, apply numeric masking to numeric columns
     4. Count stats: masked_values (total non-NaN cells in masked columns across all sheets), unique_entities (len of text_mapping)
     5. Return (masked_sheets, {"text": text_mapping, "numeric": numeric_mapping}, {"masked_values": count, "unique_entities": unique_count})
